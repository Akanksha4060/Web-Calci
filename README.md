# Web-Calci
This is a web-based calculator application built using HTML, CSS, Java, JSP, and Servlets.

## Features
- Basic arithmetic operations: addition, subtraction, multiplication, and division.
- User-friendly interface with a responsive design.
- Error handling for invalid inputs and operations.

## Technologies Used
- **HTML**: For creating the structure of the web pages.
- **CSS**: For styling the application.
- **Java**: For backend logic and calculations.
- **JSP**: For server-side rendering of the web pages.
- **Servlets**: For handling client requests and responses.
